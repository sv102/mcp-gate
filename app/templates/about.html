v0.1.0 MCP Protocol Transport — Streamable HTTP + SSE + OAuth 2.0. Claude.ai, Cursor, Windsurf, Continue, Cline connect natively. OAuth token bound to agent.
App-level authentication — three modes: basic (bcrypt + HMAC cookie), proxy (Authentik/Keycloak), none. Login page, first-time setup wizard.
Agents management — Claude, ChatGPT, Gemini, Cursor, Windsurf, Continue, Cline, Open WebUI, Custom. Per-agent command sets, rate limits, allowed hosts.
Deny command sets — type deny with highest priority. Authorization: (host_allow ∩ agent_allow) - (host_deny ∪ agent_deny).
Parameterized commands — variables with regex validation, defaults, shell character blocking.
Secrets Vault — Fernet-encrypted, $SECRET{id} substitution, output scrubbing.
Host Setup Instructions — auto-generated useradd, SSH keys, sudoers whitelist.
Import/Export — paste JSON or drag-and-drop .json for hosts, agents, command sets, secrets.
Audit — CRUD logging, quick-filter chips, advanced filters, JSON/CSV export, WebSocket real-time.
SSH security — managed known_hosts (TOFU), Ed25519 key generation.
Appearance — 6 themes, glassmorphism, custom backgrounds, custom logo/favicon.
Dashboard — stat cards, host/agent tiles with ping, clickable 24h/7d metrics.
Notifications — Telegram bot and SMTP email alerts.
i18n — English and Russian, full coverage.
v0.0.6Initial public release. Hosts, whitelist SSH, approvals (4 modes), rate limiting, dry run, web UI, audit log.
