🔑 Secrets Vault

How it works: Add a key → link to hosts → use $SECRET{id} in commands whitelist.
MCP Gate substitutes the value before SSH execution and scrubs it from the response. Claude never sees the real key.

No secrets. Add the first key or import.

Usage examples in commands

# In a command set or whitelist host:
curl -s -H "Authorization: Bearer $SECRET{grafana-token}" http://grafana:3000/api/dashboards

# Multiple secrets in one command are supported:
curl -s -u "$SECRET{elastic-user}:$SECRET{elastic-pass}" http://elastic:9200/_cluster/health

MCP Gate substitutes values, runs SSH, and scrubs keys from the response.
